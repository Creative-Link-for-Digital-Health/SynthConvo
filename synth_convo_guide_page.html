SynthConvo - Cloning & Usage Guide

This guide provides instructions for cloning and running the SynthConvo repository on your local machine. Make sure you have Node.js and Git installed before proceeding.

1. Cloning the Repository

  1. Open a terminal window on your computer.
  2. Navigate to the directory where you want to store the project.
  3. Run the following command: 
    git clone https://github.com/Creative-Link-for-Digital-Health/SynthConvo.git
  4. Navigate into the project folder: 
    cd SynthConvo

2. Installing Dependencies

  1. Ensure you have Node.js installed: node -v
  2. Install project dependencies using: 
    npm install

3. Running the Application

  1. To start the app in development mode, run: 
    npm run dev
  2. The application will likely open in your browser at http://localhost:3000 or similar.

4. Usage Overview

  • The app generates realistic conversation examples using predefined personas and scenarios.
  • Use the web interface to select or customize a persona, then initiate a conversation thread.
  • Outputs can be copied or exported for testing, demos, or study purposes.
